Detailed Description

Provides a String object to aid the development with different compilers.

Unfortunately the C++ string object is not stable, which means the standard string object <std::string> cannot be linked if the compiler version does not match. To overcome this nuisance this Object provides a string wrapper which is compatible with <std::string>

Examples
edgedetect_cuda.cpp, and edgedetect_opencl.cpp.
