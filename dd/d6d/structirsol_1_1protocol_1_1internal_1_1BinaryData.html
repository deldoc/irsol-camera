Detailed Description

template<typename T, std::uint8_t N>
struct irsol::protocol::internal::BinaryData< T, N >

Represents a binary data object in the protocol.

Template Parameters
TThe type of the elements in the binary data.
NThe dimensionality of the binary data. (e.g. 2D binary data would have N=2)
Note
This class only provides move-semantics, and does not provide copy-semantics. This is to make sure that no unnecessary copies are made while working with this data object. For this reason, members of this class are non-const, in order to allow move-semantics to be used.
