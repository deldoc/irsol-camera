Detailed Description

Definitions and utilities for incoming and outgoing protocol messages.

This header provides the core types, enums, and helper functions related to the representation and handling of protocol messages exchanged in the irsol system.

It defines:

  • Enumerations for distinguishing kinds of incoming (InMessageKind) and outgoing (OutMessageKind) messages.
  • Variant types (InMessage, OutMessage) that represent all supported message types.
  • String conversion functions for message kinds and messages.
  • Compile-time and runtime utilities to identify and check message types.

The incoming messages cover Assignments, Inquiries, and Commands. The outgoing messages cover Success acknowledgments, Binary data buffers, Black & White images, Color images, and Error reports.
